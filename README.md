# 또박또박 — 말하면 글자로

귀가 어두우신 분과의 대화를 돕는 **음성 → 글자** 화면입니다.
가운데 큰 버튼을 누르고 또박또박 말하면, 한국어 음성인식으로 큰 글씨가 화면에 나타납니다.
**"말 다 했어요"** 를 누르면 그때까지의 말이 한 문장으로 고정됩니다.

## 열기

👉 **https://greencar-uxd.github.io/grandfather/**

## 특징

- **단일 정적 HTML 한 파일**로 동작 (Web Speech API, `ko-KR`)
- 어르신용 **큰 글씨 · 고대비**, 글씨 크기 조절 (가− / 가＋, 마지막 크기를 기억)
- 말하는 중에는 실시간으로, **"말 다 했어요"** 로 문장 완성·고정
- **"화면 지우기"** 로 처음 상태로
- 문자(SMS) 발송 기능 없음 — **순수 음성 → 글자** 버전

## 권장 환경

- **컴퓨터 · 안드로이드: 크롬(Chrome)**, **아이폰: 사파리(Safari)**
- 처음 열 때 **마이크 권한 허용** 이 필요합니다.
- 음성 인식은 인터넷 연결과 브라우저의 Web Speech API 지원이 필요합니다.

## 배포

GitHub Pages — `main` 브랜치 루트의 `index.html` 한 파일로 동작합니다.
(`.nojekyll` 파일은 Jekyll 처리를 건너뛰어 정적 파일을 그대로 서빙하기 위한 것입니다.)
