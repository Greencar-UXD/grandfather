LG TV 연결 테스트

또박또박 음성 리모컨 — 우리 TV가 실제로 반응하는지 확인하는 페이지예요.
  1. 이 컴퓨터와 TV가 같은 와이파이인지 확인
  2. TV 리모컨 → 설정 → 네트워크에서 IP 주소 보기 (예: 192.168.0.12)
  3. 아래에 IP 넣고 연결 → TV 화면의 ‘허용’ 누르기
  4. 🔔 알림 보내기 눌러서 TV에 글자가 뜨면 성공!
아직 연결 안 됨
처음 연결할 때만 TV에 “접속 허용” 팝업이 떠요. 리모컨으로 ‘허용’을 누르면 끝.
 
이 파일은 컴퓨터(윈도우·맥) 브라우저에서 열어 주세요.
안 되면(연결 오류 등) 화면을 캡처해서 알려주시면 같이 고쳐요.
